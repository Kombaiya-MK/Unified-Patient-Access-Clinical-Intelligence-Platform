# User Story - US_028

## Story ID
   * US_028

## Story Title
   * Clinical Document Upload Interface

## Description
  * As a patient or staff member, I want an intuitive drag-and-drop document upload interface with progress tracking, so that I can easily submit medical records for processing.

## Acceptance Criteria
  1. **Given** I access the document upload page (SCR-008), **When** I drag PDF/PNG/JPG/DOCX files into the upload zone or click "Browse Files" to select them, **Then** the system validates file types (accepts PDF, PNG, JPG, JPEG, DOCX only), checks file size (<10MB per file, <50MB total per upload session), shows upload progress bar with percentage and estimated time remaining, uploads files to temporary storage (cloud storage bucket or local filesystem), creates ClinicalDocuments table entry with status="Uploaded", document_type (Lab Results/Imaging/Prescription/Insurance Card/Other from dropdown), uploaded_by_user_id, patient_id, original_filename, file_path, uploaded_at timestamp, and displays success confirmation with thumbnail preview of uploaded documents.

## Edge Cases
   * What happens when file exceeds size limit? (Display error: "File [name] exceeds 10MB limit. Please compress or split the file.")
   * How are duplicate uploads prevented? (Check file hash, warn "This file appears to be already uploaded. Upload anyway?" with Yes/No)
   * What if upload fails midway due to network issue? (Support resume from last chunk uploaded, show "Resume Upload" button)

> **Wireframe Details (SCR-008):**
> - **Upload zone**: Dashed border box with cloud upload icon, "Drag files here or click to browse" text
> - **File type selector**: Dropdown above upload zone (Lab Results, Imaging, Prescription, Insurance Card, Other)
> - **Progress indicators**: List of uploading files with progress bars (0-100%), file size, cancel button per file
> - **Uploaded files section**: Thumbnail gallery below upload zone, each showing preview, filename, file size, document type tag, "View" + "Delete" buttons
> - **Supported formats note**: Small text "Accepted: PDF, PNG, JPG, DOCX (max 10MB each)"
> - **Drag overlay**: When dragging files over page, show highlighted overlay "Drop files to upload"
