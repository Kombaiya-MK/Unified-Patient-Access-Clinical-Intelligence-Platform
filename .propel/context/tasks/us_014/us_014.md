# User Story - US_014

## Story ID
   * US_014

## Story Title
   * Appointment Rescheduling Functionality

## Description
  * As a patient, I want to reschedule my existing appointments through my dashboard, so that I can change appointment times if my schedule conflicts arise.

## Acceptance Criteria
  1. **Given** I am a patient viewing my upcoming appointment on the dashboard (SCR-002), **When** I click "Reschedule" button and select a new available time slot, **Then** the system updates the appointment date/time, sends updated confirmation email with new PDF, triggers calendar sync update (Google/Outlook), logs the change to audit log, and displays success message "Appointment rescheduled successfully for [new date/time]".

## Edge Cases
   * What happens when I try to reschedule within 2 hours of appointment time? (Display error: "Cannot reschedule appointments within 2 hours of start time. Please call the office.")
   * How many times can an appointment be rescheduled? (Allow up to 3 reschedulesper appointment, track reschedule_count in database)
   * What if new slot becomes unavailable during rescheduling? (Use optimistic locking, display "Slot no longer available" and reload slots)
