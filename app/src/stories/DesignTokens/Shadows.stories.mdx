# Shadows & Elevation

Progressive shadow depth for visual hierarchy. Higher levels indicate elements closer to the user in z-space.

## Elevation Guidelines

| Level | z-index Range | Use Case |
|-------|--------------|----------|
| 0 | 0 | Inline elements, no elevation needed |
| 1 | 1–10 | Cards, list items, subtle elevation |
| 2 | 10–100 | Dropdowns, popovers, menus |
| 3 | 100–1000 | Modals, drawers, overlays |
| 4 | 1000–5000 | Tooltips stacked on modals |
| 5 | 5000+ | Critical alerts, emergency banners |

---

## Usage

```css
/* CSS Custom Property */
.card {
  box-shadow: var(--shadow-level-1);
}

.modal {
  box-shadow: var(--shadow-level-3);
}

.interactive:focus-visible {
  box-shadow: var(--shadow-focus-ring);
}
```

```scss
// SCSS Variable
.card {
  box-shadow: $shadow-level-1;
}

.modal {
  box-shadow: $shadow-level-3;
}
```
