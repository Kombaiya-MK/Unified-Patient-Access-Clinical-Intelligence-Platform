## Usage

```css
/* CSS Custom Property */
.card {
  padding: var(--spacing-lg);
  margin-bottom: var(--spacing-base);
  gap: var(--spacing-sm);
}
```

```scss
// SCSS Variable
.card {
  padding: $spacing-lg;
  margin-bottom: $spacing-base;
  gap: $spacing-sm;
}
```
