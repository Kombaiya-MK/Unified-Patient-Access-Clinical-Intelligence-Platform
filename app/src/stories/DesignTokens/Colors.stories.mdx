# Color Palette

All color tokens for the Unified Patient Access Clinical Intelligence Platform.
Colors meet **WCAG 2.1 AA** contrast requirements for healthcare accessibility.

---

## Usage

```css
/* CSS Custom Property */
.button-primary {
  background-color: var(--color-primary-600);
  color: var(--color-text-inverse);
}
```

```scss
// SCSS Variable
.button-primary {
  background-color: $color-primary-600;
  color: $color-text-inverse;
}
```
