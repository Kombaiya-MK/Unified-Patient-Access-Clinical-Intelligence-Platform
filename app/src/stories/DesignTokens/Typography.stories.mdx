## Usage

```css
/* CSS Custom Property */
.heading {
  font-family: var(--typography-font-family-heading);
  font-size: var(--typography-font-size-3xl);
  font-weight: var(--typography-font-weight-bold);
  line-height: var(--typography-line-height-snug);
}
```

```scss
// SCSS Variable
.heading {
  font-family: $typography-font-family-heading;
  font-size: $typography-font-size-3xl;
  font-weight: $typography-font-weight-bold;
  line-height: $typography-line-height-snug;
}
```
